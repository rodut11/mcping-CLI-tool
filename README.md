# mcping-CLI-tool
CLI tool to ping minecraft servers inspired from kgurchiek/Minecraft-Ping-API

# How to build?
```bash
gcc -o ping ping.c
```
# How to use?
```bash
./ping <server ip> <port> [protocol]
```
Default protocol is 763 (1.20.1)

 - aternos servers (need to ping it multiple times)
